# Library Management System

A console-based Library Management System developed in Java.

## Features

- Add new books
- View all books
- Search books by ID, Title, or Author
- Add new members (students)
- Issue books to members (14-day borrowing period)
- Return books
- View currently borrowed books
- Persistent storage (data saved in CSV files)

## Technologies Used

- Java (OOP Concepts)
- Collections (ArrayList, HashMap)
- File Handling (CSV)
- java.time.LocalDate
